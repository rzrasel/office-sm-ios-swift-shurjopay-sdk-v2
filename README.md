## Installation

ShurjopaySdk is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby_podInit
pod init
```

```ruby_shurjoPaySdk
pod 'ShurjopaySdk'
```
